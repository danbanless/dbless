02 / 대량메일발송

대량 메일 발송

엑셀 명단에서 수신자를 불러와 개인화된 메일을 Outlook으로 한 번에 발송합니다.

설치 파일 다운로드

ZIP 파일을 받아 압축을 풀고, mailsender.exe를 실행하세요.
별도 설치 과정 없이 바로 사용할 수 있습니다.

Windows 전용 · Outlook 필요 · 사내 네트워크 동작
다운로드 (.7z, 11MB)

실행 환경 요구사항

  • Windows 10 이상
  • Microsoft Outlook 설치 및 계정 로그인 상태
  • ZIP 파일 내 3개 파일이 같은 폴더에 있어야 함

파일 구성

mailsender/
  ├── mailsender.exe   ← 실행 파일
  ├── index_v2.html   ← UI 화면
  └── tinymce/         ← 에디터 (폴더째 필요)

주요 기능

엑셀 명단 불러오기

xlsx 파일을 올리면 수신자 목록이 자동으로 채워집니다.

개인화 본문

{1}~{25} 자리표시자로 수신자마다 다른 내용을 삽입합니다.

발송 전 미리보기

각 수신자에게 보내지는 메일을 한 명씩 확인할 수 있습니다.

첨부파일 지원

공통 첨부파일을 전원에게 또는 개인별로 다르게 첨부합니다.

사용 방법

01

ZIP 압축 해제

다운로드한 ZIP 파일을 원하는 폴더에 압축 해제합니다.
세 파일(mailsender.exe, index_v2.html, tinymce/)이 같은 폴더에 있어야 합니다.

02

mailsender.exe 실행

exe를 더블클릭하면 브라우저가 자동으로 열리며 발송 화면이 나타납니다.

03

수신자 명단 불러오기

엑셀 파일을 업로드하거나 직접 붙여넣기로 수신자 목록을 입력합니다.
첫 번째 열이 이름, 두 번째 열이 이메일 주소입니다.

04

제목 및 본문 작성

본문에 {1}, {2} 등을 넣으면 각 수신자의 열 데이터로 자동 치환됩니다.
예: 안녕하세요, {1}님.

05

미리보기 확인 후 발송

수신자별로 최종 메일을 미리보기로 확인한 뒤 [전체 발송] 버튼을 누릅니다.
Outlook이 자동으로 메일을 발송합니다.

주의사항

Outlook이 실행 중이고 계정이 로그인된 상태여야 합니다.
세 파일은 항상 같은 폴더에 있어야 합니다. 파일을 분리하면 작동하지 않습니다.
사내 보안 정책에 따라 exe 실행이 차단될 수 있습니다. IT 부서에 문의하세요.

# TSB(팀 스케쥴 보드) 사용설명서

> 팀장 더블부킹을 막기 위한 팀 공용 캘린더입니다. 바탕화면에 상주하는 위젯(TSBWidget.exe)이며, 실행하면 바로 데이터가 동기화됩니다.

![TSB 기능 소개](tsb-기능소개-인포그래픽.png)

---

## 1. TSB가 뭔가요?

TSB(Team Schedule Board)는 우리 팀 일정을 한 화면에서 등록·공유하는 위젯입니다.

- 팀장 일정을 최상단에 별도로 보여줘서, "그 시간에 팀장님 다른 일정 있는 거 몰랐다"는 상황을 막습니다.
- 팀원 각자 컴퓨터에서 실행해도 같은 데이터를 봅니다 — 공유폴더의 데이터를 실시간으로 읽고 씁니다.
- 설치 과정이나 명령어 입력 없이, `TSBWidget.exe` 더블클릭 한 번으로 바탕화면에 위젯이 뜨고 바로 사용할 수 있습니다.

## 2. 시작하기

1. `TSBWidget.exe`를 더블클릭합니다.
2. 바탕화면에 위젯이 나타나면 바로 사용 시작 — 브라우저나 별도 접속 주소는 필요 없습니다.
3. 위젯을 켜둔 상태로 두면 다른 팀원 PC와 자동으로 데이터가 동기화됩니다.

> 사외망(재택 등)에서는 공유폴더에 접근할 수 없어 데이터가 로컬에 임시로 저장될 수 있습니다. 사내망 복귀 후 다시 실행하면 공유폴더 데이터로 자동 전환됩니다. *(재택 동작 방식은 실제 배포 전 확인 필요)*

## 3. 화면 구성

| 영역 | 설명 |
|---|---|
| 상단 드롭다운 | 지금 "누구로 접속했는지" 선택 — 새 일정은 이 이름으로 등록됩니다 |
| 🌙 | 라이트/다크 모드 전환 (OS 다크모드 자동 감지) |
| 🔒 나만 보기 | 내 일정만 필터링해서 봅니다 |
| 🔕 | 팀장 일정 1시간 전 알림 켜기/끄기 |
| 🔅 슬라이더 | 위젯 투명도 조절 |
| ⚙ 팀 관리 | 팀원 추가/삭제, 팀장 지정, 이름·이메일 수정, 백업/복원 |
| + 새 일정 | 일정 등록 |
| 일간 · 주간 · 월간 | 뷰 전환 (◀ ▶ 로 이전/다음, 오늘 로 즉시 복귀) |

## 4. 핵심 기능

- **팀장 레인 분리** — 팀장이 참석하는 일정은 항상 최상단에 별도 표시되고 인디고색 "팀장" 배지가 붙습니다.
- **충돌 사전 경고** — 저장 전에 같은 참석자·겹치는 시간대 일정이 있으면 경고가 뜨고, "그래도 등록" 또는 "시간 수정" 중 선택합니다.
- **유형별 색상 구분** — 미팅(인디고) · 교육(테라코타) · 기타(그레이). 교육 유형은 대상자 전체 선택과 장소 입력을 지원합니다.
- **등록자 권한** — 본인이 만든 일정만 수정·삭제할 수 있습니다. 남의 일정은 볼 수만 있습니다.
- **동시 저장 안전장치** — 여러 명이 동시에 저장해도 락 파일로 순서를 보호해 데이터가 깨지지 않습니다.
- **공휴일 자동 표시** — 공공데이터(한국천문연구원) 연동으로 달력에 공휴일이 자동 표시됩니다.
- **Excel 백업/복원** — "⚙ 팀 관리"에서 실제 .xlsx 파일로 백업 다운로드·복원이 가능합니다 (관리자 전용).

## 5. 사용 방법

1. **사용자 전환** — 상단 드롭다운에서 본인 이름 선택.
2. **뷰 전환 / 날짜 이동** — 일간·주간·월간 버튼, ◀ ▶, 오늘 버튼 활용.
3. **새 일정 등록** — `+ 새 일정` 클릭 → 유형·제목·날짜·시간(10분 단위)·참석자 입력. 팀장을 참석자로 체크하면 팀장 레인에 노출됩니다.
4. **충돌 확인** — 겹치는 일정이 있으면 저장 전 경고가 뜹니다. "그래도 등록" 또는 "시간 수정" 중 선택하세요.
5. **일정 확인/수정** — 보드의 일정 칩을 클릭하면 상세 보기. 본인 일정이면 수정·삭제 버튼이 나타납니다.

## 6. 관리자 전용 기능

`⚙ 팀 관리`에서 아래 항목은 **관리자로 전환했을 때만** 가능합니다.

- 팀장 지정
- 팀원 이름·이메일 교정
- 팀원 삭제
- Excel 백업/복원 (복원 시 공유 데이터 전체가 교체되니 주의하세요)

팀원 추가는 관리자가 아니어도 누구나 할 수 있습니다.

## 7. 알려진 제한사항

> ⚠️ 아래 항목은 브라우저 프로토타입 시점 기준이며, TSBWidget.exe 실제 배포 전에 최신 상태로 확인·수정이 필요합니다.

- 바탕화면 고정(Win+D에도 남는 것), 트레이 상주 여부
- Excel(.xlsx) 형식 자체로의 실시간 저장 여부 (또는 JSON 기반 동기화 유지 여부)
- 재택(사외망) 시 동기화 동작 방식

## 8. 자주 묻는 질문

**Q. 위젯을 껐다 켜면 일정이 사라지나요?**
A. 아니요. 일정은 공유폴더(또는 사외망일 땐 로컬)에 저장되므로 위젯을 껐다 켜도 유지됩니다.

**Q. 다른 사람이 등록한 일정을 내가 수정할 수 있나요?**
A. 아니요. 본인이 등록한 일정만 수정·삭제할 수 있습니다.

**Q. 저장하다가 "락 획득 실패" 오류가 뜨면?**
A. 다른 사람이 거의 동시에 저장 중이라는 뜻입니다. 잠시 후 다시 시도하면 됩니다.

**Q. 재택근무 중에도 쓸 수 있나요?**
A. 실행은 가능하지만, 사외망에서는 다른 PC와 실시간 동기화되지 않을 수 있습니다. *(정확한 동작은 배포 전 확인 필요)*

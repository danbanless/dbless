06 / 명찰생성기

명찰 생성기

엑셀 명단과 템플릿 하나로 명찰을 A4에 자동 배치해 pptx로 출력합니다. 설치 없이 exe 하나로 실행.

실행 파일 다운로드

exe 파일 하나를 받아 더블클릭하면 바로 실행됩니다.
별도 설치, Python 없이도 동작합니다.

Windows 전용 · 단일 EXE · 인터넷 연결 불필요
다운로드 (.7z)

실행 환경 요구사항

  • Windows 10 이상
  • Microsoft PowerPoint (출력 확인용 — 생성 자체는 불필요)
  • 명단 파일: xlsx 형식, 1행 헤더 + 2행부터 데이터

파일 구성

원하는 폴더/
  └── 명찰생성기.exe   ← 이것 하나만

주요 기능

템플릿 자동 생성

크기를 입력하면 {{이름}} 등 플레이스홀더가 들어간 pptx 템플릿을 바로 만들어줍니다.

A4 자동 배치

여백·간격을 입력하면 A4 한 장에 몇 개 들어가는지 즉시 계산해 미리보기로 보여줍니다.

가로형 자동 전환

명찰 가로 크기가 A4 세로 용지를 초과하면 가로형 A4로 자동 전환합니다.

재단선 커스터마이즈

재단선 색상·두께·선 종류(실선·점선·파선 등)를 자유롭게 설정합니다.

이미지·디자인 유지

템플릿에 넣은 배경 이미지, 로고가 각 명찰에 그대로 복사됩니다.

컬럼명 = 플레이스홀더

엑셀 헤더 이름이 곧 {{컬럼명}}으로 사용됩니다. 컬럼 수·명칭 제한 없음.

사용 방법

01

탭① — 템플릿 생성

명찰 크기(mm)를 입력하고 [템플릿 생성 및 저장]을 누릅니다.
저장된 template_90x60.pptx를 PowerPoint에서 열어 디자인을 편집합니다.

02

템플릿 편집 (PowerPoint)

플레이스홀더 위치·폰트·크기 조정, 배경 이미지·로고 삽입 후 저장합니다.
플레이스홀더는 반드시 {{이름}} 처럼 중괄호 두 겹으로 작성하세요.

03

탭② — 파일 선택

편집한 template.pptx와 명단 data.xlsx를 각각 불러옵니다.
xlsx 1행은 헤더(컬럼명), 2행부터 실제 데이터입니다.

04

레이아웃 설정

A4 여백(상·하·좌·우)과 명찰 간격을 mm 단위로 입력합니다.
우측 미리보기 캔버스에서 배치가 실시간으로 확인됩니다.

05

명찰 생성

출력 파일명을 입력하고 [명찰 생성]을 누릅니다.
output/파일명.pptx가 exe와 같은 폴더에 생성됩니다. PowerPoint에서 열어 인쇄하세요.

주의사항

템플릿 내 이미지는 그룹화(Ctrl+G)하지 마세요. 겹치는 것은 괜찮습니다.
명찰 크기가 A4(최대 297×210mm)를 초과하면 생성이 차단됩니다.
재단 시 명찰이 지정 치수보다 크게 잘리지 않도록 재단선 안쪽으로 칼을 대세요.

출력 크기 오차 주의

프린터 드라이버 설정에 따라 실제 출력물이 설정 크기보다 작게 나올 수 있습니다.
반드시 테스트 출력 후 자로 실측하세요.

오차가 있을 경우 아래 공식으로 템플릿 크기를 보정해 재생성하세요.

보정 크기 = 목표 크기 × (목표 크기 ÷ 실측 크기)

예) 목표 90mm, 실측 86mm
→ 90 × (90 ÷ 86) ≒ 94.2mm → 95mm로 템플릿 재생성

프린터 속성에서 배율을 100%로 고정하면 오차를 줄일 수 있습니다.
(PowerPoint 인쇄 화면 → 프린터 속성 → 배율 100% 고정)
